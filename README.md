# blog-server

## 源码结构

```
info.lostred.blog     
├── annotation                          // 自定义注解
├── aspect                              // 切面(业务层日志、控制层校验)
├── config                              // 第三方框架配置
├── controller                          // 控制层
├── dto                                 // 传输数据包装
├── entity                              // 业务实体类
├── handler                             // 处理器
├── mapper                              // 数据访问层
├── service                             // 业务层
├── util                                // 工具类
└── vo                                  // 表现实体类
```
